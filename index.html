ブライトコーブ・ライブ

ライブモジュールは、デスクトップとモバイルデバイスの両方にライブイベントをブロードキャストするために使用されます。イベント情報を Live モジュールに入力すると、オンサイトのエンコーダに接続する一連のエンコーダ設定が提供されます。トランスコーディングはすべてクラウドで実行されるため、ハードウェアベースのトランスコーダは必要ありません。

より複雑なユースケースと統合の場合は、Live API を使用することもできます。
