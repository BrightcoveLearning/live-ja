このトピックでは、ライブモジュールを使用してビデオクリップをスケジュールする方法を学習します。

コントロールルームでのクリップのスケジュール

  1. ライブモジュールで、静的エントリポイントを使用して新しいライブイベントを作成するか、以前に作成したイベントにアクセスします。
  2. コントロールルームで、[ スケジュールされたクリップ ] セクションまでスクロールします。
  3. [ スケジュールされたクリップを追加] をクリックして、新しいクリップをスケジュールします。
    注:の場合スケジュールされたクリップパネルはで利用できません制御室、Brightcoveサポートに連絡して有効にしてください。
  4. モーダルウィンドウが表示されます。[ 開始日時]と [ 終了日時] を選択します。
      注:
    • スケジュールされたクリップの開始日時、終了日時と時刻は、現在または将来のストリームイベントと一致する必要があります。終了日は将来の日付である必要があります。そうしないと、クリップの作成が失敗します。
    • スケジュールされたクリップの最小時間は5分、最大時間は24時間です。
  5. クリップのデュレーションは自動的に計算されます。
  6. 名前を付けて。
  7. 簡単な説明を書いてください。（オプション）
  8. 必要に応じてタグを割り当てます。（オプション）
  9. 参照 ID を追加します。IDは、Studioアカウント全体で一意の値である必要があります。（オプション）
  10. [ クリップのスケジュール] をクリックします。
    11. スケジュールされたクリップは、終了日に達すると作成されます。一方、空のビデオアセットはメディアモジュールのプレースホルダーとして作成されます。
  11. スケジュールされたクリップを更新するには、クリップをクリックします。
  12. 必要な変更を加えます。
  13. [ クリップを更新] をクリックします。
  14. スケジュールされたクリップを削除するには、ゴミ箱アイコンをクリックします。

クリッピングページでのクリップのスケジュール

  1. コントロールルームで、左側のメニューから [ クリップを作成] をクリックします。
  2. [ スケジュールされたクリップを追加] をクリックして、新しいクリップをスケジュールします。
    注:の場合スケジュールされたクリップパネルはで利用できません制御室、Brightcoveサポートに連絡して有効にしてください。
  3. モーダルウィンドウが表示されます。[ 開始日時]と [ 終了日時] を選択します。
      注:
    • スケジュールされたクリップの開始日時、終了日時と時刻は、現在または将来のストリームイベントと一致する必要があります。終了日は将来の日付である必要があります。そうしないと、クリップの作成が失敗します。
    • スケジュールされたクリップの最小時間は5分、最大時間は24時間です。
  4. クリップのデュレーションは自動的に計算されます。
  5. 名前を付けて。
  6. 簡単な説明を書いてください。（オプション）
  7. 必要に応じてタグを割り当てます。（オプション）
  8. 参照 ID を追加します。IDは、Studioアカウント全体で一意の値である必要があります。（オプション）
  9. [ クリップのスケジュール] をクリックします。
    10. スケジュールされたクリップは、終了日に達すると作成されます。一方、空のビデオアセットはメディアモジュールのプレースホルダーとして作成されます。
  10. スケジュールされたクリップを更新するには、クリップをクリックします。
  11. 必要な変更を加えます。
  12. [ クリップを更新] をクリックします。
  13. スケジュールされたクリップを削除するには、ゴミ箱アイコンをクリックします。
  • スケジュールされたクリップの開始日と終了日は、終了日に達する前にのみ更新できます。
  • スケジュールされたクリップパネルに赤いエントリが表示される場合があります。これらは失敗したスケジュールです。スケジュールされたクリップは、VODの開始時間と終了時間に含まれないキューポイントなど、さまざまな理由で失敗する可能性があります。
