はじめに

Avaya Rooms Systemsは、オンラインで他の人と会うことができるビデオ会議アプリケーションです。より多くのオーディエンスにリーチするために、ブライトコーブライブを活用してミーティングをストリーミングして、他のユーザーが見られるようにすることができます。

ステップ

ブライトコーブライブでミーティングを主催するには、次の手順に従います。

  1. Video Cloud Studio で、ライブモジュールを使用してライブイベントを作成します。
    放送完了後にブロードキャストを保存する場合は、[ 完了したらイベントをビデオアセットに変換 ] チェックボックスをオンにしてください。

  2. ライブイベントを作成したら、コントロールパネルからストリーミング RTMP URL とストリーミングキーをコピーします。

    • ストリーミング RTMP URL- ストリーミングエンドポイント URL（エンドポイント URL は Brightcove Live Control Room にあります）
    • ストリーミングキー- ストリーム名（ブライトコーブでは、ストリーム名は常に「生きている」）
  3. Web サイト/イントラネットページにこのライブイベントのプレーヤーを公開して埋め込み、ページの URL を書き留めます。

ストリーミングセッションの設定

ストリーミング RTMP URL とストリーミングキーを使用して、Avaya コーデックユニットで設定する必要があります。

  1. Avaya GUI で、[ 設定] > [ストリーミング] に移動します。

    (オプション) Avaya ウェブサイトで、[ 管理者設定] > [ユーティリティ] > [ストリーミング] の順に選択します。

  2. ストリーミング RTMP URL を挿入します。
  3. ストリーミングキーを挿入します。

ストリーミングセッションの開始/停止

ストリーミングセッションを開始または停止するには、次の手順を実行します。

  1. ホームページ > その他 > ストリーミングに移動します。
  2. [ ストリーミング ] ボタンをクリックします。これにより、ライブストリームが開始されます。
  3. RTMP ストリーミングはデフォルトで有効になっています。これを無効にするには、Avaya プラットフォームで [設定] > [ 詳細設定] > [ユーティリティ] > [ストリーミング] の順に進みます。
