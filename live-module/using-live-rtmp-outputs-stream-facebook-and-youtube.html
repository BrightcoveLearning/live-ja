ライブRTMP出力を使用してFacebookとYouTubeにストリーミング

このトピックでは、ライブモジュールで RTMP 出力を使用して Facebook と YouTube にライブイベントをストリーミングする方法について説明します。

はじめに

ライブモジュールは、デスクトップとモバイルデバイスの両方にライブイベントをブロードキャストするために使用できます。イベント情報を Live モジュールに入力すると、オンサイトのエンコーダに接続する一連のエンコーダ設定が提供されます。通常、ライブストリームは Brightcove Player の内部で公開されます。ライブモジュールコントロールルームページでは、追加の RTMP 出力 URL を入力できます。RTMP 出力設定は、ソーシャルメディアサイトにライブストリームを送信するために使用できます。RTMP 出力時間は、イベント時間に対して請求されることに注意してください。 このトピックでは、簡単なライブイベントを作成する手順について説明し、RTMP 出力を使用して Facebook と YouTube にライブイベントをストリーミングする方法について説明します。

  • サーバーサイド広告（SSAI）を含む広告は、RTMP出力にストリーミングされません。広告なしのストリームが代わりに使用されます。
  • RTMP 出力は、ライブジョブごとに 5 つの出力に制限されます。この制限には、エラーとしてマークされた接続が含まれます。

新しいイベントを作成する

新しいイベントを作成するには、次の手順に従います。

  1. ライブモジュールを開きます。
  2. [イベントの管理]ページから、[ イベントの作成ボタンまたはクリックライブイベントを作成するまたライブチャンネルを作成する左側のナビゲーションで。「ライブイベントの作成」または「ライブチャンネルの作成」ページが開きます。
  3. [ イベント名]と、必要なカスタムフィールドを入力します。
  4. ライブインジェストプロファイルを選択します。
  5. 完了したらイベントをビデオアセットに変換を選択して、VOD バージョンのライブ動画を Video Cloud アカウントに保存します。
  6. [ 詳細オプション ] をクリックし、必要に応じて詳細オプションを設定します。
  7. [ ストリーミング開始] をクリックして、ライブイベントを開始します。Control Room ページが開き、エンコーダで使用されるストリーミングエンドポイント（RTMP URL）やストリーム名など、ライブイベントの詳細が表示されます。
    注:ストリームの処理には約 1 分かかります。
    注:デフォルトでは、新しいイベントが作成されてから 30 分以内にエンコーダを起動する必要があります。
  8. エンコーダからストリーミングを開始すると、ライブストリームが Control Room ページに表示されます。

ライブイベントが開始されると、ライブストリームをFacebookやYouTubeに送信できます。

注:ライブストリームが完了したら、エンコーダが閉じられているか、停止していることを確認してください。データが送信されていない場合でも、エンコーダとブライトコーブの間の接続を開いたままにしておくと、予期しない接続料金が発生する可能性があります。

Facebookライブへのストリーミング

Facebook   Liveは、     Facebookのライブビデオストリーミング機能で、  ライブイベントを配信することができます。会社のページまたは個人プロフィールを通じてオーディエンス。ライブ動画は、 ライブイベント中、およびイベントの終了後に、個人のニュースフィードに表示されます。 

注:Facebook のライブストリーミングの詳細については、 Facebook のドキュメントを参照してください。

ライブモジュールでライブストリーミングが開始されたら、以下の手順に従って Facebook でライブストリームを使用します。

  1. Facebookアカウントにログインします。
  2. 新しい投稿を作成し、ライブビデオオプションをクリックします。
  3. Facebookは、コンピュータのカメラとマイクへのアクセスを促すことがあります。このトピックでは、イベントに外部デジタル一眼レフカメラを使用しているため、[ 接続] をクリックします。
  4. Facebook はサーバー URL とストリームキーを表示します。これらの値は Live モジュールで使用されます。
  5. サーバーURLをクリップボードにコピーします。
  6. ライブモジュールで、[コントロールルーム] ページの [ RTMP 出力 URL ] フィールドにサーバー URL を貼り付けます。
  7. Facebook からクリップボードにストリームキーをコピーします。
  8. ライブモジュールで、[ RTMP 出力 URL ] フィールドの末尾にストリームキーを貼り付けます。RTMP 出力の URLフィールドは、ストリームキーが追加された Facebook サーバーの URLで、すべて 1 つの値になります。
    注:RTMP URL で現在サポートされている唯一のフォーマットはですrtmps://host:port/application/streamName。
  9. [レンディションの選択] ドロップダウンからレンディションを選択し、[RTMP 出力を追加] をクリックします。
  10. RTMP セクションの下で、接続が成功したことを確認します。
  11. ライブストリームが Facebook に表示されることを確認します。Facebook のタイトルと説明を入力します。
  12. [ ライブに移動] をクリックします。

ライブストリームは Facebook で開始されます。

[ 完了 ] をクリックして Facebook のライブストリームを終了します。イベントが完了したときも、エンコーダを停止することを忘れないでください。

YouTubeにストリーミングする

YouTube には、ライブストリーミング動画を視聴者に配信する機能があります。ライブ配信を初めて開始する前に、YouTube チャンネルを有効にする必要があります。

注:ライブストリームを初めて有効にするには 、最大 24 時間かかる場合があります。有効にすると、 ストリームは即座にライブ配信されます。
注:YouTube ライブストリーミングの詳細については、 YouTube のドキュメントを参照してください。

ライブモジュールでライブストリーミングが開始されたら、以下の手順に従って YouTube でライブストリームを使用します。

  1. YouTube アカウントにログインします。
  2. https://studio.youtube.com の YouTube Studioのページに移動します。
  3. 左側のナビゲーションで、[ その他の機能] > [今すぐライブストリーム] をクリックします。
  4. YouTube では、パソコンのカメラとマイクへのアクセスが求められることがあります。このトピックでは、イベントに外部デジタル一眼レフカメラを使用しているため、[ 接続] をクリックします。
  5. YouTube にサーバー URL とストリーム名/キーが表示されます。これらの値は Live モジュールで使用されます。
  6. サーバーURLをクリップボードにコピーします。
  7. ライブモジュールで、[コントロールルーム] ページの [ RTMP 出力 URL ] フィールドにサーバー URL を貼り付けます。
    注:RTMP URL で現在サポートされている唯一のフォーマットはですrtmps://host:port/application/streamName。
  8. YouTube からストリーム名/キーをクリップボードにコピーします。
  9. ライブモジュールで、[ RTMP 出力 URL ] フィールドの末尾にストリーム名/キーを貼り付けます。[ RTMP 出力 URL ] フィールドは、ストリーム名/キーが追加された YouTube サーバー URLになり、すべて 1 つの値になります。
  10. [レンディションの選択] ドロップダウンからレンディションを選択し、[RTMP 出力を追加] をクリックします。
  11. RTMP セクションの下で、接続が成功したことを確認します。
  12. ライブストリームが YouTube に表示されることを確認します。

ライブ配信は YouTube で開始されます。

[YouTube Studio に戻る] をクリックし、エンコーダーを停止してライブストリームを終了します。配信が完了すると、公開動画が自動的に作成され、後で視聴できるようにアップロードされます。
