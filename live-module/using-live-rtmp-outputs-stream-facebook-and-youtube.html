ライブRTMP出力を使用してFacebookやYouTubeにストリーミングする

このトピックでは、ライブモジュールで RTMP 出力を使用して、ライブイベントを Facebook や YouTube にストリーミングする方法について説明します。

はじめに

Live モジュールは、ライブイベントをデスクトップデバイスとモバイルデバイスの両方にブロードキャストするために使用できます。 Live モジュールにイベント情報を入力すると、オンサイトのエンコーダに接続する一連のエンコーダ設定が提供されます。 通常、ライブストリームは Brightcove Player の内部で公開されます。 ライブモジュールの [コントロールルーム] ページでは、追加の RTMP 出力 URL を入力できます。 RTMP 出力設定を使用して、ライブストリームをソーシャルメディアサイトに送信できます。 RTMP 出力時間はイベント時間に対して課金されることに注意してください。 このトピックでは、簡単なライブイベントを作成する手順について説明し、RTMP 出力を使用して Facebook や YouTube にライブイベントをストリーミングする方法について説明します。

注:サーバーサイド広告（SSAI）を含む広告は、RTMP 出力にはストリーミングされません。 代わりに、広告なしのストリームが使用されます。

新しいイベントの作成

新しいイベントを作成するには、次の手順を実行します。

  1. Live モジュールを開きます。
  2. 「イベントの管理」ページで「 イベントを作成 」ボタンをクリックするか、左側のナビゲーションで「 ライブイベント を作成」または「ライブチャンネル を作成」をクリックします。 「ライブイベントの作成」または「ライブチャンネルの作成」ページが開きます。
  3. 「 イベント名 」と必須のカスタム・フィールドを入力します。
  4. ライブ取り込みプロファイル を選択します。
  5. ライブ ビデオの VOD バージョンを Video Cloud アカウントに保存するには、[完了したらイベントをビデオアセットに変換 ] を選択します。
  6. [ 詳細オプション ] をクリックし、必要に応じて詳細オプションを設定します。
  7. ライブイベント を開始するには、[ストリーミング の開始] をクリックします。 [Control Room] ページが開き、エンコーダが使用する ストリーミングエンドポイント（RTMP URL） や ストリーム名 など、ライブイベントの詳細が表示されます。
    注:ストリームの処理には約 1 分かかります。
    注：デフォルトでは、新しいイベントが作成されてから 30 分以内にエンコーダを起動する必要があります。
  8. エンコーダからストリーミングを開始すると、ライブストリームが Control Room ページに表示されます。

ライブイベントが開始されると、ライブストリームは Facebook や YouTube に送信されます。

注：ライブ配信の完了時に、エンコーダが閉じているか停止していることを確認してください。 データが送信されていない場合でも、エンコーダとブライトコーブ間の接続を開いたままにしておくと、予期しない接続料金が発生する可能性があります。

Facebookライブへのストリーミング

Facebook Liveは、Facebookのライブビデオストリーミング機能で、会社のページや個人プロフィールを通じてオーディエンスにライブイベントをブロードキャストできます。 ライブ動画は、ライブイベント中、およびイベント終了後に個人のニュースフィードに表示されます。

注:Facebook ライブストリーミングの詳細については、Faceb ook のドキュメントを参照してください 。

ライブモジュールでライブストリーミングを開始したら、以下の手順に従って Facebook でライブストリームを使用します。

  1. Facebookアカウントにログインします。
  2. 新しい投稿を作成し、 ライブビデオ オプションをクリックします。
  3. Facebookは、お使いのコンピュータのカメラとマイクへのアクセスを要求することがあります。 このトピックでは、イベントに外付けデジタル一眼レフカメラを使用しているので、[ 接続 ] をクリックします。
  4. Facebookは、 サーバーURL と ストリームキー を表示します。 これらの値は Live モジュールで使用されます。
  5. サーバー URL をクリップボードにコピーします。
  6. ライブモジュールで、[コントロールルーム] ページの [ RTMP 出力 URL ] フィールドに サーバー URL を貼り付けます。
  7. Facebookからクリップボードに ストリームキー をコピーします。
  8. ライブモジュールで、 RTMP 出力 URL フィールドの最後に ストリームキーを 貼り付けます。 RTMP 出力 URL フィールドは、 ストリームキー が追加された Facebook サーバーの URL で、すべてが 1 つの値になります。
  9. [レンディションの選択] ドロップダウンから レンディションを選択し 、[ RTMP 出力を追加 ] をクリックします。
  10. RTMP セクションの下で、接続が成功したことを確認します。
  11. ライブストリームが Facebook に表示されることを確認します。 Facebook のタイトルと説明を入力します。
  12. [ ライブに移動 ] をクリックします。

ライブ配信は Facebook で開始されます。

「 完了 」をクリックして、Facebook のライブストリームを終了します。 イベントが完了したときも、エンコーダを停止することを忘れないでください。

YouTube へのストリーミング

YouTube では、視聴者にライブストリーミング動画を配信できます。 初めてライブ配信を開始する前に、YouTube チャンネルを有効にする必要があります。

注:ライブ配信を初めて有効にするには最大で 24 時間かかる場合があります。 有効にすると、ストリームは即座にライブ配信されます。
注:YouTube ライブストリーミングの詳細については、YouTub e のドキュメントを参照してください 。

ライブモジュールでライブストリーミングを開始したら、以下の手順に従って YouTube でライブストリームを使用します。

  1. YouTube アカウントにログインします。
  2. YouTubeスタジオ のページへ https://studio.youtube.com。
  3. 左側のナビゲーションで、[ その他の機能] > [今すぐライブ配信 ] をクリックします。
  4. YouTube では、パソコンのカメラとマイクへのアクセスを求めるメッセージが表示されることがあります。 このトピックでは、イベントに外付けデジタル一眼レフカメラを使用しているので、[ 接続 ] をクリックします。
  5. YouTube には、 サーバーの URL と ストリーム名/キー が表示されます。 これらの値は Live モジュールで使用されます。
  6. サーバー URL をクリップボードにコピーします。
  7. ライブモジュールで、[コントロールルーム] ページの [ RTMP 出力 URL ] フィールドに サーバー URL を貼り付けます。
  8. YouTube の ストリーム名/キーを クリップボードにコピーします。
  9. ライブモジュールで、 RTMP 出力 URL フィールドの末尾に ストリーム名/キーを 貼り付けます。 RTMP 出力 URL フィールドは、 ストリーム名/キー が付加された YouTube サーバーの URL で、すべて 1 つの値になります。
  10. [レンディションの選択] ドロップダウンから レンディションを選択し 、[ RTMP 出力を追加 ] をクリックします。
  11. RTMP セクションの下で、接続が成功したことを確認します。
  12. ライブ配信が YouTube に表示されることを確認します。

ライブ配信は YouTube で開始されます。

[ YouTube Studio に戻る ] をクリックし、エンコーダを停止してライブ配信を終了します。 配信が完了すると、公開動画が自動的に作成され、後で視聴できるようにアップロードされます。
