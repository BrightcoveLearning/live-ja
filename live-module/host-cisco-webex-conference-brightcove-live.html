はじめに

Cisco Webexは、オンラインで他の人と会えることができるビデオ会議アプリケーションです。より多くのオーディエンスにリーチするために、 Brightcove Liveを活用してミーティングをストリーミングして、他のユーザーが見られるようにすることができます。

ステップ

Brightcove Liveでミーティングを主催するには、次の手順に従います。

  1. サイト管理者として Cisco Webex プラットフォームにログインします。

  2. サイト管理またはコントロールハブを使用して、他のストリーミングサービスへのライブストリームアクセスを有効にします。。

  3. サイト管理またはコントロールハブを使用して、個々のユーザーのライブストリーミングを有効にします。

    [ ユーザー ] に移動し、更新するユーザーアカウントを選択します。ユーザーの [ 高度なユーザー設定とトラッキングコード]セクションをクリックし、[ ライブストリーミングを有効にする ] チェックボックスをオンにします。

  4. ブライトコーブのビデオクラウドスタジオにログインします。

  5. ライブモジュールに移動し、ブライトコーブのライブイベントを作成します。詳細については、『Live Module を使用したライブイベントの作成と管理』ドキュメントを参照してください。

    放送完了後にブロードキャストを保存する場合は、[ 完了したらイベントをビデオアセットに変換 ] チェックボックスをオンにしてください。

  6. このイベントを作成すると、次の情報が提供されます。

    • エンドポイント (RTMP URL) -ストリーミング URL
    • ストリーム名 - 値が常に生きているストリーミングキー
  7. このライブイベントのプレーヤーを公開し、ウェブサイトまたはイントラネットページに埋め込みます。ページ URL をクリップボードにコピーします。

  8. Cisco Webex アプリケーションに戻ります。Webex ミーティングを開始またはスケジュールします。これを行うには、縦の 3 ドットアイコンを選択し、[ ライブストリーミングを開始] を選択します。

  9. [ その他のストリーミングサービス情報 ] ダイアログで、次のように入力します。

    • ストリーミングサービス :使うBrightcove
    • ターゲットストリームリンク : Brightcove Liveの [ エンドポイント (RTMP URL) ] の値を使用します。
    • ターゲットストリームキー :使用alive (これは常にこの値です)

  10. それだおまえは生きてる！
