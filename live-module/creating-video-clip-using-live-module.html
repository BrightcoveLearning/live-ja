ライブモジュールを使用したビデオクリップの作成

このトピックでは、Live モジュールを使用してビデオクリップを作成する方法を学習します。
 

概要

Live モジュールは、完了したイベントからビデオクリップを作成する機能を提供します。 ライブイベントの進行中にクリップを作成することもできます。 クリップを作成する理由はいくつかあります。

  • ソーシャルメディアに投稿できる短いティーザークリップを作成するには
  • 投稿する動画の長さを制限するには（ほとんどのプラットフォームは動画の長さを制限します）
  • あなたは、短い動画がソーシャルメディアでより良いパフォーマンスを発見しました

クリッピングは、すべてのタイプのライブイベントで実行できます。また、ライブストリーム 中 およびライブストリーム 後に Live API または Live モジュールを使用して行うこともできます。

注記

  • 従来の取り込みプロファイルがライブクリッピングをサポートしなくなりました。 ライブ配信からビデオクリップを作成するには、アカウントで動的配信が有効になっていることを確認し、動的配信取り込みプロファイルを使用します。
  • クリップは、イベント終了後7日以内に作成することができます。 その後、[ クリップを作成 ] ボタンは無効になります。 イベントが 24 時間を超える場合は、過去 24 時間のみが切り取られます。
  • 定期的なイベントの場合:イベント後 7 日間にイベントが再アクティブ化された場合、クリッピングは最新のイベントのコンテンツでのみ使用できます。 前回のイベントが無効になってから 7 日以内であっても、前回のイベントからクリップを作成することはできません。
  • 新しいライブストリームのチャンネルを停止して再起動すると、クリッピング用の以前のライブストリーム情報が上書きされることがあります。
    • 。
  • エンコーダがストリームの SMPTE タイムコードを送信している場合、クリップは フレーム精度になります。そうでない場合は、2 フレーム以内で正確になります。 可能な限り迅速にクリップを提供するために、セグメント精度のクリップが最初に作成され、利用可能になったらすぐにフレーム精度のクリップに置き換えられます。
  • ストリームへの開始時間が 10 秒未満のクリップは作成できません。
  • ライブストリームをクリッピングして作成された動画では、メディア共有が失敗します。
  • AES で暗号化されたライブストリームからクリップを作成することはできません。
定期的なイベントを作成すると、Media モジュールでそのイベント用のビデオが自動的に作成されます。 イベントが再発するたびに必要になるため、このビデオを削除 しないでください 。

クリップエディタのナビゲート

クリップエディターを開くには、Live モジュールを開き、イベント名をクリックし、左側のナビゲーションで Create Clip をクリックします。 クリップエディタが開きます。

クリッピングインターフェイスのすべての時間は、ブライトコーブが動画データを受信した UTC 時間に基づいていることに注意してください。 これは、VOD ウィンドウを超過したストリームを考慮するために必要な調整です (24 時間以上実行)。 ビデオがそのウィンドウを超えた場合、プレーヤーから戻った時間が確実にクリップの生成に使用できなくなりました。

クリップエディタは、次のように構成されています。

  • A- プレビュープレーヤー
  • B- 再生コントロールボタン
  • C- クリップ時間
  • D- クリッピングボタン
  • E- クリップのプレビューボタン
  • F-保存ボタン
  • G- プレイヘッド
  • H- フォーカスタイムライン
  • I- コンテキストタイムライン
  • J- ズームボタン
  • K- クリップの一覧

クリップの作成

クリップを作成する場合、選択した ライブ取り込みプロファイルの最高 品質のレンディションがクリップのソースとして使用されます。 クリップのレンディションを作成するときに、アカウントのデフォルトのアップロード取り込みプロファイルが使用されます。

クリップを作成する手順は、次のとおりです。

  1. クリッピングボタンまたはショートカットキーを使用して、クリップの開始点と終了点を設定します。
  2. [ 保存 ] をクリックします。
  3. クリップビデオのプロパティを入力します。
    • Clip Name ：クリップの名前
    • 短い説明 -クリップの説明
    • タグ -クリップに関連付けるタグ
    • リファレンス ID -クリップに割り当てるリファレンス ID
    • ソース広告区切りコンテンツを削除 -このオプションを選択すると、プログラムコンテンツのみを保持し、クリップ内で発生した広告区切りを削除します。
      注:[ソース広告の区切りを削除] チェックボックスは、ストリームが SSAI が有効になっている場合にのみ表示されます。

    必須のカスタムフィールドも表示されます。

  4. [ クリップを作成 ] をクリックします。 クリップのトランスコーディング処理が開始されます。 クリップは、アカウントのデフォルトの取り込みプロファイルを使用してトランスコードされます。

クリップエディタの下に、すべてのクリップの一覧が表示されます。 クリップ名 をクリックすると、別のブラウザタブにあるメディアモジュールのクリップが開きます。 「 ソーシャルに公開 」をクリックして、ソーシャルモジュールでクリップを開き、ソーシャルメディアのアカウントに公開します。 Social モジュールを使用したビデオの公開については、「 ソーシャルメディアプラットフォームへのビデオの公開 」を参照してください。

プレイヤをプレビュー

プレビュープレーヤーは、クリップのプレビューに使用されます。 イベントが進行中で、クリップエディタを開くと、クリップの再生が開始されます。クリップは、終了前の約 3 セグメントの「安全なライブポイント」から再生されます。 すべてのブラウザが自動再生をサポートしているわけではないことに注意してください。

再生コントロールボタン

再生コントロールボタンは、プレビュープレーヤー内のクリップの再生を制御するために使用します。

ボタン 説明
ソースビデオの先頭からビデオ再生を開始します。
再生ヘッドを 15 秒前にジャンプします。
再生ヘッド位置からビデオ再生を開始します。
再生ヘッドを 15 秒前にジャンプします。
 

クリップ時間

クリップタイムには、再生ヘッドの現在の UTC 時間とクリップ可能な時間が表示されます。 UTCリンクをクリックすると、現地時間表示に切り替わります。

クリッピングボタン

クリッピングボタンは、ソースビデオの所望の部分をクリップするために使用されます。

ボタン 説明
ビデオのポイントでキューにジャンプします
クリップの開始点を設定します。 フォーカスタイラインの先頭にあるハンドルをドラッグして開始点を設定することもできます。
クリップの終了点を設定します。 また、フォーカスタイラインの末尾にあるハンドルをドラッグして終了ポイントを設定することもできます。
動画のキューアウトポイントにジャンプします。

キューインポイントとキューアウトポイントは、クリップの開始ポイントと終了ポイントです。 これらの各ポイントは、ドラッグ可能なハンドルでフォーカスタイムライン上に表示され、ポイント間の時間は緑色で塗りつぶされます。 また、コンテキストタイムラインでは、クリップ空間を示す緑色の長方形の左右の端として表されます。

プレビュークリップ

クリップを再生するには、クリップのプレビューボタンをクリックします。

再生ヘッド

再生ヘッドは、プレイヤーが再生している時間の指標です。 フォーカスタイムラインでは、再生ヘッドが赤い線で表され、コンテキストタイムラインでは赤い丸と赤い縦線で表されます。

フォーカスタイムラインで、再生ヘッドをドラッグしてビデオをスクラブすることができます。 フォーカスのタイムラインをダブルクリックすると、再生ヘッドがそのポイントに移動します。 再生ヘッドは、キューイン/キューアウトボタンまたはキーボードショートカットを使用して、キューインポイントとキューアウトポイントを設定するために使用することもできます。

フォーカスタイムライン

フォーカスタイムラインは、インターフェイスの最上位のタイムラインです。

再生ヘッドは、プレイヤーが再生している時間の指標です。 フォーカスタイムラインでは、再生ヘッドが赤い線で表され、上に赤い円が表示されます。

 

コンテキストタイムライン

コンテキストタイムラインは、クリップエディタの一番下のタイムラインです。 コンテキストタイムラインには、ズームレベル、クリップの選択した領域、再生ヘッドの位置、現在のフォーカスビューポートに関係なく、ビデオのデュレーション全体が表示されます。 タイムラインは赤い線で示されます。

[ズーム] ボタン

ズームボタンは、フォーカスのタイムライン上でズームインまたはズームアウトするために使用されます。

ボタン 説明
フォーカスタイムラインを拡大表示します。 コンテキストタイムラインのフォーカスビューポート (白いボックス) には、フォーカスタイムラインに表示されているクリップの領域が表示されます。
フォーカスタイムラインを縮小します。 コンテキストタイムラインのフォーカスビューポート (白いボックス) には、フォーカスタイムラインに表示されているクリップの領域が表示されます。

クリップの一覧

クリップエディタの下に、クリップの一覧が表示されます。 [ Publish to Social ] をクリックして、Brightcove Social を使用してクリップをソーシャルサイトの 1 つにパブリッシュします。 Social モジュールを使用したビデオの公開については、「 ソーシャルメディアプラットフォームへのビデオの公開 」を参照してください。

キーボードショートカット

キーボードショートカットは、ビデオからクリップを再生したり作成したりするためにも使用できます。 次のキーボードショートカットを使用できます。

再生コントロール

このキーを押す これを行うには
スペース、k、ダウン 再生/一時停止
私 キューインに移動
O キューアウトに移動
うp プレビューを再生
j、左 1フレーム戻る
J、シフト+左 ジャンプ1秒バック
l, 右 1 フレーム前方にジャンプ
L、シフト+右 1秒前にジャンプ
s 開始へジャンプ
e 最後までジャンプ/ライブ

タイムラインコントロール

このキーを押す これを行うには
i キューインを設定
o キューアウトを設定
+ ズームイン
- 縮小表示
u タイムラインを再生ヘッドにパン
？ ヘルプモーダルを開く
