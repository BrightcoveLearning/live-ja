ズーム会議をBrightcoveLiveに統合する

Brightcove Live Zoomの統合により、BrightcoveLiveを使用してZoom会議を簡単にブロードキャストできます。

前提条件

ズームマーケットプレイスで

ブライトコーブライブとのズーム統合は、ズームマーケットプレイスのブライトコーブライブコネクトアプリによって管理されます。これは、ライセンスされた所有者または管理者レベルのZoomアカウントで使用するためのアカウントレベルのアプリです。Zoom アカウントに、統合をサポートする正しい権限があることを確認してください。

ブライトコーブの動画クラウドで

統合を開始する前に、イベントが Zoom からストリーミングされているかどうかを追跡するカスタムメタデータフィールドが必要です。これにより、セッション間の統合の状態が追跡されます。つまり、イベントが現在 Zoom ミーティングにリンクされている場合です。カスタムメタデータフィールドの追加方法の詳細については、次のドキュメントを参照してください。

  1. Video Cloud Studio > 管理者 > ビデオフィールド > カスタムフィールドの追加に移動します。
    2. 。
  2. カスタムメタデータフィールドの表示名を記述します。
  3. [ 内部名 ] フィールドには、次のように記述する必要があります。 bc_meeting_id
    カスタムフィールドに外部ズームIDまたはデータを追加しないでください。
  4. 書く説明カスタムメタデータフィールド用。

ベストプラクティス

最良の結果を得るには、以下の指示に従ってください。

  1. 使用始めるとやめるホストのズームプレーヤーではなく、コントロールルームのブロードキャストボタン。
  2. 単一のアカウントで複数のZoomミーティングを同時にストリーミングすることは避けてください。

ズームミーティングの作成

  1. まず、 Zoom アカウントにアクセスし、Zoom ミーティングを作成します。Zoom ミーティングの作成方法については、 Zoom のドキュメントを参照してください。
  2. Zoom で、 [設定] > [ ミーティング (詳細設定)] > [ ライブストリーミングミーティング]の順に選択します。
  3. Zoom アカウント設定でカスタムライブストリーミングサービスが有効になっていることを確認します。

ZoomアカウントとBrightcoveLiveの統合

Zoom アカウントを Brightcove Live と統合するには、次の手順に従います。

  1. ビデオクラウドスタジオにログインします。
  2. ナビゲーションヘッダーの [ Admin ] ボタンをクリックします。
  3. ドロップダウンメニューから [ 統合]オプションを選択します。
  4. [ 統合を追加]ボタンをクリックして、1 つまたは複数の Zoom アカウントをブライトコーブビデオクラウドにリンクします。
  5. Zoom アカウントへのアクセスを要求するポップアップウィンドウが表示されます。[ 承認]ボタンをクリックします。
  6. Zoom 認証情報を入力して、アカウントを統合します。

ブライトコーブのライブイベントへのズームインテグレーションの追加

Brightcove ライブイベントに Zoom インテグレーションを追加するには、次の手順に従います。

  1. Video Cloud の Liveモジュールに移動します。
  2. [ ライブイベントの管理]ページの [ イベントを作成 ] ボタンをクリックします。ライブイベントの作成方法の詳細については、「ライブモジュールを使用したライブイベントの作成と管理」を参照してください。
  3. 「統合」セクションに到達するまで、ライブイベントの詳細を通常どおり入力します。
  4. [ 統合の追加 ] ボタンをクリックします。
  5. アクティブなZoom会議とスケジュールされたZoom会議のリストを見ることができます。
  6. ドロップダウンメニューをクリックします。
  7. 会議を選択します。この例では、テストドキュメント会議を選択します。
    イベントに一度に追加できる統合は 1 つだけです。統合をクリアするには、[ 統合の削除 ] ボタンをクリックします。複数のアカウントを使用する場合は、[ 管理] > [統合] パネルでこれを行うことができます。
  8. ライブイベントを作成する準備ができたら、[ イベントを作成]ボタンをクリックします。
  9. コントロールルームには、Zoom から会議をストリーミングするためのタブ付きオプションがあります。統合ソースを使用したくない場合は、左上の[URLの取り込み]タブに戻ります。
  10. Zoomミーティングが最近作成された場合は、同期ボタン。
    同期しても会議が表示されない場合は、会議が表示されていることを確認してください予定 Zoomで、スケジュールされている会議が30未満であること。
  11. Zoomミーティングアカウントにアクセスしてください。
  12. 青い [ 開始 ] ボタンをクリックして、会議を開始します。
    ズームの制限により、ズーム会議をBrightcove Liveにブロードキャストする前に、ズーム会議を開始する必要があります。
  13. 戻るブライトコーブのライブコントロールルームで、[ ブロードキャストを開始]ボタンをクリックします。
  14. 最後の手順の後、コントロールルームで Zoom Meeting を見ることができます。
    • ズームミーティング中はいつでも、ブロードキャストを停止および再開できます。ただし、アクティブにブロードキャストしていない場合、ギャップがイベントの再接続時間を超えると、イベントがタイムアウトすることがあります。
    • ブライトコーブでのズームブロードキャストは、約 20 秒遅れます。
    • Zoom ミーティングが 20 秒経過してもストリーミングが開始されない場合は、[ ブロードキャストを停止] をクリックしてから [ ブロードキャストを開始]をもう一度クリックします。
    • Zoomストリームはサードパーティのサービスであるため、SSAIなどの一部の高度なライブ機能がサポートされていない場合があります。

ズーム統合のアンインストール

ビデオクラウドからインテグレーションを削除する

ブライトコーブのアカウントから Zoom インテグレーションを削除するには:

  1. ビデオクラウドスタジオにログインします。
  2. ナビゲーションヘッダーの [ Admin ] ボタンをクリックします。
  3. ドロップダウンメニューから [ 統合]オプションを選択します。
  4. ゴミ箱アイコンをクリックします。
  5. 確認を求められたら、[ 確認]ボタンをクリックします。

Zoom アカウントからブライトコーブライブコネクトを削除するには

Zoom に接続すると、Zoom アカウントからブライトコーブライブコネクトアプリを完全に削除できます。

  1. Zoom アカウントにログインし、 Zoomアプリマーケットプレイスにナビゲートします。
  2. 管理 >インストール済みアプリをクリックするか、ブライトコーブアプリを検索します。
  3. ブライトコーブアプリをクリックします。
  4. [ アンインストール] をクリックします。
