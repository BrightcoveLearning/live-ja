ライブ設定の管理

このトピックでは、Brightcove Live の設定を管理する方法について説明します。

ライブ設定では、フィルスレートとライブ認証情報を管理できます。

注:Brightcove Live で使用されるサーバーサイド広告設定は、サーバーサイド広告設定の管理の一部として構成されます。 詳しくは、「 サーバーサイド広告設定の構成 」を参照してください。

充填スレートの管理

フィルスレートは、ライブ配信の広告切れにギャップがある場合に提供できるコンテンツです。 スレートは、「Be right back」メッセージやその他の表示したいコンテンツを提供するために使用できます。 すべての Zencoder 入力形式 がサポートされています。

注：充填スレートは Live API を使用して管理することもできます。 詳細については、「 Brightcove Live API: サーバーサイド広告挿入 (SSAI) 」を参照してください。

塗りつぶしスレートを追加するには、次の手順に従います。

  1. Studioヘッダーの［ 管理］>［ライブ設定 ］リンクをクリックします。
    注:[ADMIN] メニューにアクセスするには、アカウント管理者である必要があります。
  2. [ + 塗り潰しスレートを追加 ] をクリックします。
  3. 「 名前 」を入力します。
  4. スレタイの ソースURL を入力します。
  5. [ 保存 ] をクリックします。 スレートの摂取プロセスが始まります。

塗りつぶしスレートを削除するには、塗りつぶしスレートの横にある削除ボタン () をクリックします。

資格情報の管理

認証情報は、Live API と Live モジュールの両方で使用されます。 新しい Live Video Cloud 認証情報を作成する場合は、まず API 認証認証情報を作成する必要があります。 作成されたクライアント ID とクライアントシークレットは、Live Video Cloud の認証情報を作成するときに使用されます。 FTP、S3、または RTMP 認証情報を作成する場合、API 認証認証情報は必要ありません。

S3、FTP、または RTMP 認証情報を作成する場合は、 ステップ 5 に進みます 。 Live Video Cloud の認証情報を作成するには、まず API 認証情報を作成します。

  1. Studioのヘッダーにある［ ADMIN ］リンクをクリックします。
    注:[ADMIN] メニューにアクセスするには、アカウント管理者である必要があります。
  2. [ API 認証 ] リンクをクリックします。 API認証ページが開き、現在のクライアント登録が表示されます。
  3. [ 新規アプリケーションの登録 ] をクリックして、新しい API 認証情報を作成します。 次のAPIを選択する必要があります。
    注意:API認証資格証明の作成の詳細は、「API認証資格 証明の管理 」を参照してください。
  4. クレデンシャルが作成されたら、 クライアント ID とクライアント シークレット の値を保存します。 これらは、Live Video Cloud の認証情報を作成するときに必要になります。
  5. Studioヘッダーの［ 管理］>［ライブ設定 ］リンクをクリックします。
  6. + [資格情報の追加 ] をクリックします。
  7. ラベル を入力します。 これは、資格情報を参照するために内部的に使用される名前です。
  8. タイプ を選択:
    • FTP -VOD アセットを FTP の場所に保存するために使用します（API 使用）。
    • RTMP ：エンコーダで使用する RTMP クレデンシャルを作成するために使用します。
    • S3 -VOD アセットを Amazon S3 アカウントに保存するために使用します（API 使用）。
    • ビデオクラウド -VOD アセットをビデオクラウドアカウントに保存するために使用します (API 使用)。
  9. [ 公開キー]/[ユーザー名 ] に、クレデンシャルのタイプに応じて次のように入力します。
    • FTP-FTP サイトのユーザー名を入力します。
    • RTMP ：エンコーダが認証に使用するユーザ名を入力します。
    • S3-S3 アカウントのユーザー名を入力します。
    • Video Cloud -API 認証資格情報用に生成された クライアント ID を入力します。
  10. [ 秘密鍵/パスワード ] には、クレデンシャルのタイプに応じて次のように入力します。
    • FTP-FTP サイトのパスワードを入力します。
    • RTMP ：エンコーダが認証に使用するパスワードを入力します。
    • S3-S3 アカウントのパスワードを入力します。
    • Video Cloud -API 認証資格情報用に生成された クライアントシークレット を入力します。
  11. [ 保存 ] をクリックします。

資格情報が資格情報のリストに表示されます。 資格情報を削除するには、資格情報の横にある削除ボタン（）をクリックします。
