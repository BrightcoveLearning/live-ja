シミュレートされたライブイベントの作成

ライブイベントの作成と実行のコストとワークフローに投資する前に、視聴者の関心を評価するテストを試してみてください。 このサンプルでは、ライブストリーミングイベントをシミュレートする方法を示します。

はじめに

ライブストリーミングとVODを区別するのは、時間の経過です。 コンテンツストリームは時間に関連付けられ、すべての視聴者がそのタイムラインに同期されます。

このサンプルでは、Video Cloud 再生リスト内の VOD コンテンツと JavaScript を使用して、ライブストリーミングイベントのこの効果をシミュレートし、再生を開始するタイミングに関係なく、すべての視聴者が再生リスト内で同じポイントを維持します。

以下に 2 人のプレイヤーが表示され、次にこれがどのように作成されたかについての説明が表示されます。

選手たち

Brightcove ラーニングサービス ( @rcrooks1969 ) による CodePen のペン シミュレーションライブイベント をご覧ください。

使用する API メソッド

  • プレイリストを取得するCatolog.getPlayList ()
  • プレイリストをプレイヤーにロードするCatalog.loadPlayList ()
  • playlist.autoadvance（0） は、ビデオ間の一時停止なしでオートアドバンスにプレイリストを設定します
  • playlist.repeat（真） は、終了後に再生リストを繰り返すように設定します
  • player.playlist（） は、プレイリストからビデオデータを取得します
  • PlayList.currentItem は、再生リスト内の特定のビデオを現在のビデオとして設定する
  • Player.currentTime（） は、ビデオ内の指定された時間にシークします
