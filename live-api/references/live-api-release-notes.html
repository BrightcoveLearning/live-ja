ライブ API リリースノート

これは、ライブ API の変更の概要です。

2019-05-13

  • ライブジョブを作成するための 出力 フィールドに color_full_width が追加されました。

2019-05-06

  • 進行中のライブストリームに 広告メタデータを追加する サポートが追加されました。

2019-01-28

  • RTMP 出力 のサポートが追加されました。

2019-01-21

  • 新しい入力プロトコルの サポートが追加されました（MPEG2-TS 入力を可能にする）。
